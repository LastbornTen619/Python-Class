# PySide6 paso a paso

Esta carpeta contiene ejemplos cortos para aprender interfaz grafica con Python usando PySide6.

PySide6 es la version oficial de Qt para Python. Se ve mas moderna que Tkinter y permite crear ventanas, botones, formularios y aplicaciones de escritorio con una estructura clara.

## Orden recomendado

Lee y ejecuta los archivos en este orden:

1. `01_ventana_basica.py`
2. `02_boton_basico.py`
3. `03_layout_vertical.py`
4. `04_formulario_simple.py`
5. `05_menu_basico.py`

## Instalar la libreria

Abre una terminal dentro de `Python-Class` y ejecuta:

```bash
pip install PySide6
```

Si usas Windows y tienes varias versiones de Python, puede ser mejor:

```bash
py -m pip install PySide6
```

## Como ejecutar un ejemplo

Ejemplo:

```bash
py ejemplos/04-pyside6/01_ventana_basica.py
```

O tambien:

```bash
python ejemplos/04-pyside6/01_ventana_basica.py
```

## Recomendacion para estudiar

1. Lee primero los comentarios del archivo.
2. Ejecuta el programa.
3. Cambia el titulo, el texto o el tamano de la ventana.
4. Vuelve a ejecutar para ver el resultado.

## Antes de subir tus cambios

Haz esto siempre:

1. Lee el archivo para entenderlo.
2. Ejecuta el archivo para comprobar que funciona.
3. Si modificaste algo, ejecutalo otra vez.
4. Luego recien haz `git add`, `git commit` y `git push`.
