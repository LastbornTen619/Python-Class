# Interfaces Modernas Con PySide6

Este ejemplo usa `PySide6`, una libreria moderna para crear interfaces graficas en Python.

## Instalar la libreria

```bash
pip install PySide6
```

## Ejecutar el ejemplo

```bash
python ejemplos/07-interfaces-pyside6/interfaz_moderna.py
```

## Elementos que incluye

1. Ventana principal.
2. Etiquetas.
3. Caja de texto.
4. Selector numerico.
5. Lista desplegable.
6. Casilla de verificacion.
7. Botones de opcion.
8. Deslizador.
9. Barra de progreso.
10. Lista.
11. Area de texto.
12. Tabla.
13. Botones.
14. Ventana emergente.
