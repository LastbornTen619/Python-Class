# Analisis De Datos Y Graficas

Estos ejemplos sirven para aprender a:

1. Importar librerias.
2. Leer datos desde archivos.
3. Mostrar datos en consola.
4. Crear graficas.
5. Leer datos desde Excel.
6. Calcular correlacion.

## Librerias necesarias

```bash
python -m pip install pandas matplotlib openpyxl
```

## Archivos de ejemplo

1. `leer_datos_y_graficar.py`: crea listas dentro del programa, las convierte en un DataFrame y dibuja una grafica.
2. `excel_y_correlacion.py`: crea un Excel de ejemplo, lo lee y calcula correlacion.

## Ejecutar los scripts

```bash
python ejemplos/07-analisis-datos/leer_datos_y_graficar.py
python ejemplos/07-analisis-datos/excel_y_correlacion.py
```
